# nhl-twitter-ts-bot
A NHL Twitter bot made in TypeScript.

## Running Locally
Make sure you set your environment variables.
```
git clone https://github.com/elementemerald/nhl-twitter-ts-bot
cp .env.template .env
npm i
npm run build
npm run start
```
